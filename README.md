# Danboruya's dotfiles

## Description

Dotfiles for each OS. Set up the new environment.

## Install

Download the repository.

```bash
cd ; git clone git@github.com:danbourya/dotfiles
```

Setup the environment.

```bash
./dotfiles/etc/setup.sh
```
